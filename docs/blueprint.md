# **App Name**: Aravalli Configurator

## Core Features:

- Material Selection: Allow users to select steel, PVC, or other material types for their furniture design.
- Furniture Customization: Enable users to customize dimensions (length, width, height), and other features of the furniture (drawers, shelves, doors) using UI elements.
- Visual Preview: Provide a real-time 3D preview of the furniture as it's being customized.
- Generative Suggestion: Suggest design improvements based on the current customization.
- Cost Estimation: Provide an estimated cost of the furniture based on selected materials and dimensions. As a tool, the application will leverage its understanding of real-time price changes, so that the estimate incorporates up-to-date material pricing and availability.
- Order Placement: Allow users to place an order for the customized furniture, including delivery options and payment.

## Style Guidelines:

- Primary color: Deep, saturated grayish blue (#468189), suggestive of industrial materials
- Background color: Very pale grayish blue (#E0E8E9), similar hue but extremely desaturated and bright for a light color scheme.
- Accent color: Moderate blue violet (#7744FF), for highlights and points of user focus
- Body and headline font: 'Inter', a sans-serif font providing a modern, neutral look suitable for a configuration tool.
- Use clean, simple line icons to represent different furniture components and customization options.
- Employ a modular grid layout to showcase furniture options and customization settings.
- Use subtle transitions and animations to provide feedback and enhance the user experience during customization.
